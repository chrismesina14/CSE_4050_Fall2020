About this Website

This website is an on-going personal project that is built on HTML and CSS. For the 'Resume' section, I used an HTML template found online which implements some JavaScript code for its animations and some features. I edited some of the template's design so that it can fit in to my personal style and preference. For the 'Contact' section, I used a Linktree account so that you can find all my social media and professional account information there. For my 'Portfolio' section, I am still in the process of creating and brainstorming the ideas for that page. Also, I plan on improving this website's capabilities more by implementing JavaScript functions so that I can put more new design, animations, and features.
